## Architecture (settled)

Certificate is **exactly two SNF triples**: `certK` for ∂_k, `certM` for M.
Everything else is derived in Lean or closed by `native_decide`.

`HomologyQuotientCert ∂_k ∂_{k+1}` (in `Cultivar/Homology/Basic.lean`):
- `certK : CertificateSNF ∂_k`              — U, V, D for ∂_k
- `hCC : ∂_k * ∂_{k+1} = 0`                 — closed by `native_decide`
- `M : Matrix (Fin (n - certK.r)) (Fin p) ℤ` — defined as bottom (n-r) rows of V⁻¹·∂_{k+1}
- `certM : CertificateSNF M`                 — U', V', D' for M

`CertificateHomology X k` wraps `HomologyQuotientCert (boundaryK X k) (boundaryK X (k+1))`.

Sage's role: untrusted oracle. It produces witnesses (U, V, D, U', V', D'); Lean verifies via `native_decide`. The whole point is that we never trust Sage's conclusion — only check its work.

## Deferred (post-Friday)

**Bridge theorem** in `Cultivar/Homology/Bridge.lean` (new file). Statement:
$$\ker(\partial_k) / \mathrm{im}(\partial_{k+1}) \cong \bigoplus_i \mathbb{Z}/d_i\mathbb{Z} \oplus \mathbb{Z}^{\beta_k}$$
where $d_i$ are the nonzero non-unit diagonal entries of $D'$ and $\beta_k$ counts zero diagonal entries of $D'$.

Proof sketch: cycle basis $Z$ (last $n-r$ columns of $V$) gives $\ker(\partial_k) \cong \mathbb{Z}^{n-r}$; under this ID, $\mathrm{im}(\partial_{k+1}) = \mathrm{im}(M)$; so the quotient is $\mathrm{coker}(M) \cong \mathrm{coker}(D')$ (unimodular change of basis); read off via Mathlib's PID structure theorem.

For Friday: state as `axiom` (or `theorem … := by sorry`) in `Bridge.lean`. Tactic uses it; kernel sees a hole but the metaprogramming pipeline is end-to-end exercised.
